Tic·Tac·Toe

by Risclover

Instructions expand_more

This rendition of Tic Tac Toe requires 2 players to play on the same computer. Players take turns placing their respective markers on the game board. The first person to get 3 markers in a row horizontally, vertically, or diagonally wins!

This version of Tic Tac Toe has some settings that you can set before you play, but if you want to start a game quickly without bothering with these, you can simply press the Start Game button to get started.

  1. Either press Start Game to get started immediately, or customize your game via the settings before clicking Start Game.
    1. Starting without customizing the settings results in the following default settings:
      • Player 1 Name: Player 1
      • Player 1 Marker: X
      • Player 2 Name: Player 2
      • Player 2 Marker: O
      • Rounds: 1

Settings

Rounds

  • Single Round: Play a single round.
  • # of Rounds: Pick # of rounds to play.
  • Ongoing Rounds: Play ongoing rounds until players decide to stop.

Deciding Pieces

X always goes first.

  • Players Pick: Players decide amongst themselves who gets which piece.
    • Regular: Pick who gets "X" and who gets "O".
    • Custom: If you would like to use custom markers, you can! This game uses Font Awesome's library of free icons.
      1. Go to Font Awesome website, located at https://www.fontawesome.com/
      2. Click "Icons"
      3. The version should automatically be set to the latest version. If you use Font Awesome regularly for your own purposes, make sure that it is via the dropdown menu located in the search box.
      4. To start viewing the library without searching for anything, simply click on the Search box and press Enter.
      5. Select the "Free" filter. Pro icons will not work.
      6. Find an icon to use as your marker and click on it.
      7. When the icon information pops up, you will see a code snippet that looks similar to this: <i class="fa-solid fa-00"></i> Click on it to copy it to your clipboard.
      8. Paste the <i></i> code snippet into the text box for your player's custom marker.
  • Alternate: Pieces alternate between players each round.
  • Random: Computer chooses pieces at random at the start of each round.
