Tic·Tac·Toe

by Risclover

Instructions expand_more

This is Tic Tac Toe, the classic game that we've all played as children. Two players play against one another on one computer. Choose how many rounds you want to play, use custom markers, and be the first to 3 in a row!

Quick Start

Click the "Start Game" button at the bottom of the form without filling any details out if you want to start a quick game.

Doing so will start a 1-round game with Player 1 as X and Player 2 as O.

Custom Game

Fill in form elements for any settings you wish to customize.

  1. Fill in each player's name. Leaving these fields blank will result in the players being called "Player 1" and "Player 2".
  2. Enter how many rounds you'd like to play. Leaving this blank will result in a 1-round game.
    • Single Round: Play a 1-round game
    • Pick Rounds: Decide how many rounds you want to play. Input this number into the box that appears after making this selection.
    • Ongoing Rounds: Play up to the maximum amount of rounds possible (9,007,199,254,740,991 rounds). Click the "Stop Game" button (located beneath the game board) to end the game when you decide that you're done.
  3. Pick your markers. Leaving this dropdown menu blank will result in Player 1 as X's and Player 2 as O's.
    • Classic (X's and O's): Use the classic X and O markers. Decide who gets what via the new dropdown menus that appear.
    • Custom: Pick unique markers using the Font Awesome icons collection. For more information on how to do this, refer to the Using Custom Marker section below.

Playing the Game

  1. Once the game begins, players will take turns clicking squares to set their respective markers.
  2. The first player to get 3 markers in a row in any direction wins the round.
  3. Refer to the top box (just above the gameboard) to check to see who's turn it is.
    • First round: Whoever chose X goes first. If using custom markers, player 1 goes first.
    • Subsequent rounds: Palyers will alternate going first each round.
  4. If you fill the board with markers and neither player gets 3 in a row, the round ends in a draw. This counts as a round play regardless of neither player scoring a point.
  5. The game will end in the following circumstances:
    • Single Round: A player gets 3 in a row or the board is filled, ending in a tie.
    • Picked Rounds: Once the chosen number of rounds have been played, the player with the highest score wins.
    • Ongoing Rounds: Continues until the "Stop Game" button is pressed.
  6. When the game ends, if players have tied scores, the game will offer a tiebreaker round to break the tie.
    • Whomever wins the tiebreaker round wins the game.
    • If this round ends in another tie, the game will offer additional tiebreaker rounds until either a) one player scores, or b) players decide to quit.

Using Custom Markers

You have the option of using custom markers via Font Awesome's library of icons. If you wish to do so and aren't sure how, follow the instructions here.

  1. Select the "Custom" option in the "Game Pieces" dropdown menu, which will result in two new boxes appearing, one for each player's marker.
  2. Go to the Font Awesome website at https://www.fontawesome.com.
  3. Click "Icons"
  4. Make sure that the version is set to the latest version
  5. Use the filters to customize your search. Select "Free" so that only free icons show up. Pro icons will NOT work.
  6. When you find an icon you like, click on it. A popup window will open.
  7. You should see a code snippet in this popup window with the following format:

    <i class="fa-solid fa-heart"></i>

    Click on it to copy it to your clipboard, or highlight it and copy it yourself.
  8. Paste the code snippet into the correct player's input box.

As long as the version number is correct and the icon is a "Free" icon, this should work without any issues.
